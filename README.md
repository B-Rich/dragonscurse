Dragon's Curse Action/Adventure Platform Game:
![Screenshot01](screenshot01.png)

1. Download:  
git clone https://github.com/phoboz/dragonscurse.git  
  
2. Build on Linux:  
cd dragonscurse  
make  
  
3. a) Play on Linux:  
./dragonscurse
  
Controls:  
- Arrow key to move  
- Press [SPACE] to jump  
- Press [Left Ctrl] to attack
