Video of Game running natively on Linux for Tegra:   
http://youtu.be/cssXbu-QunE   
   
1. Download:  
git clone https://github.com/phoboz/dragonscurse   
  
2. Build on Linux:  
cd dragonscurse  
make  
  
3. a) Play on Linux:  
./dragonscurse
  
Controls:  
- Arrow key to move  
- Press [SPACE] to jump  
- Press [Left Ctrl] to attack
